Visão Geral da Landing Page

A Landing Page do Rei do Crédito é uma página de conversão otimizada para captação de leads interessados em empréstimos do Bolsa Família. A página foi desenvolvida com foco em conversão e experiência do usuário, utilizando tecnologias web modernas.

🎯 Objetivos Principais

💰 Geração de Leads

Captar interessados em empréstimos do Bolsa Família através de formulários otimizados e CTAs estratégicos.

🔗 Conversão para WhatsApp

Direcionar usuários qualificados para atendimento humanizado via WhatsApp.

🏆 Construção de Confiança

Demonstrar credibilidade através de depoimentos reais, avaliações e presença física.

✨ Características Técnicas

  • SPA (Single Page Application): Construída como aplicação React
  • Design Responsivo: Otimizada para todos os dispositivos
  • SEO Otimizado: Meta tags e structured data implementados
  • Performance: Lazy loading e otimizações de carregamento
  • Integração WhatsApp: Links diretos para atendimento
  • Analytics: Tracking de conversões e comportamento

Instruções de Build e Deploy

A landing page está otimizada para deploy em hospedagem compartilhada, especialmente Locaweb:

📋 Pré-requisitos

  • Hospedagem com suporte a Apache
  • Suporte a PHP (opcional, para compatibilidade)
  • Acesso via FTP/SFTP
  • Painel de controle da hospedagem

🚀 Processo de Deploy

# 1. Preparação do ambiente
- Acesse o painel da Locaweb
- Navegue até a pasta public_html
- REMOVA todos os arquivos existentes (limpeza)

# 2. Upload dos arquivos
- Extraia TODO o conteúdo do ZIP rei-do-credito-build
- Faça upload de TODOS os arquivos para public_html
- Certifique-se que index.html está na RAIZ

# 3. Verificação da estrutura
public_html/
├── index.html (OBRIGATÓRIO na raiz)
├── index.php (compatibilidade)
├── .htaccess (configurações)
├── assets/ (CSS/JS)
├── images/ (imagens)
└── outros arquivos...

# 4. Teste de funcionamento
- Acesse seu domínio
- Teste /test.php para diagnóstico
- Verifique todos os CTAs WhatsApp

⚙️ Configurações do Servidor

O arquivo .htaccess inclui configurações essenciais:

# SPA Routing - redirecionar para index.html
RewriteCond %{REQUEST_FILENAME} !-f
RewriteCond %{REQUEST_FILENAME} !-d
RewriteRule . /index.html [L]

# Cache Control para performance
<FilesMatch "\.(css|js|png|jpg|svg|webp|ico)$">
    Header set Cache-Control "public, max-age=31536000"
</FilesMatch>

# Compressão GZIP
AddOutputFilterByType DEFLATE text/css application/javascript

# Security Headers
Header always set X-Content-Type-Options nosniff
Header always set X-XSS-Protection "1; mode=block"
